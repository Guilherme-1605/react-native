# Comandos Úteis

- npx create-expo-app@latest --template  //cria o projeto React Native com Expo

# Pacotes instalados

- npx expo install expo-image
- npx expo install @expo/vector-icons

# Comandos Expo

- npm start (roda o projeto no expo)
- npx expo start --tunnel
